  <li>Recomendo usar o git para clonar o projeto.</li>

## ⚙️ Instalação

Após a clonagem do repositório, siga os passos a seguir:

```bash
  ### Instalar as dependências
  $ npm i
  
  ### Rodar a aplicação
  $ npm run dev
  
  ### A aplicação deve rodar na porta http://localhost:5173/
```
